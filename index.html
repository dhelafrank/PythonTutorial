Title: Understanding Data Types in Python

1. Introduction to Data Types

The importance of data types in programming lies in their role of defining the kind of data a variable can hold and the operations that can be performed on it. Data types ensure data integrity, memory allocation, and compatibility between different parts of a program. They also help catch errors and enhance code efficiency by allowing the compiler or interpreter to optimize memory usage and execution speed. By choosing appropriate data types, programmers can create robust, efficient, and reliable software.

Importance of Data Types to the computer

Data types help computers understand and manipulate different kinds of information by providing a structured way to represent and process data. Computers operate based on binary code, which is composed of 1s and 0s. Data types define how these binary values should be interpreted to represent various types of information, such as numbers, characters, and more complex structures.

For example, when you declare a variable with a specific data type like "int" (integer), the computer knows to allocate a fixed amount of memory to store whole numbers and how to perform arithmetic operations on them. Similarly, when you use a "string" data type, the computer understands that the data represents a sequence of characters rather than numerical values.

Data types also help computers ensure data validity and prevent unintended operations. If you try to perform arithmetic on a string, the data type system will flag it as an error since strings are not intended for mathematical operations. This prevents mistakes and helps programmers write accurate and efficient code.

In summary, data types serve as a bridge between the abstract concepts programmers work with and the binary language computers understand, allowing for precise manipulation and interpretation of various types of data.

  1. Integers: Imagine you're shopping online and adding items to your cart. The quantities of items you select are represented as integers. The computer uses integer data types to keep track of the number of each item.

  2. Floating-Point Numbers: When you're calculating the price of items in your cart, you might encounter fractions of a cent. Floating-point numbers are used to represent these decimal values accurately.

  3. Strings: Whenever you're entering your name, address, or a message on a website, that text is stored as strings. For instance, a website's login system uses strings to store usernames and passwords.

  4. Boolean: Consider a light switch â€“ it can be either "on" or "off". Boolean data types, which have values of either true or false, are used to represent such binary states. They're commonly used in decision-making and control structures.

  5. Dates and Times: Online calendars and appointment scheduling systems use various data types to manage dates and times. These can include integers for the day, month, and year, as well as floating-point numbers to handle time zones and fractional time portions.

  6. Arrays and Lists: In a to-do list app, the list of tasks you create is stored in an array or list data structure. Each item in the list can be a string or another data type, allowing you to organize and manage tasks effectively.

  7. Characters: When you send a text message, each letter, number, and symbol is represented by a character. These characters are encoded using specific data types, such as ASCII or Unicode, allowing computers to display and interpret text correctly.

These are just a few examples of how different data types are used in everyday scenarios, demonstrating the versatility and importance of data types in various applications.

2. Numeric Data Types

  • Integer Data Type:

    Definition

An integer data type is a fundamental data type in programming that is used to represent whole numbers. It includes both positive and negative whole numbers, as well as zero, without any decimal or fractional parts. Integers are typically stored as fixed-size binary representations in a computer's memory.

The integer data type is commonly used in scenarios where whole numbers are required, such as:

  1. Counting: Keeping track of quantities, items, or occurrences that involve discrete values. For example, the number of items in a shopping cart or the number of likes on a social media post.

  2. Indexing: Accessing elements in arrays, lists, or other data structures. The position of an element is often represented using integers.

  3. Loop Control: In programming loops (like for or while loops), integers are often used to control the number of iterations or to iterate through a range of values.

  4. Mathematical Operations: Integers are used in arithmetic operations like addition, subtraction, multiplication, and division, where the result is expected to be a whole number.

  5. Enumerations: When creating enumerated types to represent a set of distinct values, integers are commonly used to associate each value with a unique integer identifier.

In programming languages, integer data types might have different sizes (in terms of the number of bits used to store them) based on the language and the underlying architecture of the computer. Common integer data types include "int" (often 32 or 64 bits), "short" (usually 16 bits), and "long" (usually 64 bits). The choice of integer data type depends on the range of values you need to represent and the memory constraints of your application.

Operations With Integers

Here are examples of basic arithmetic operations using integers in Python:

  1. Addition (+):

    num1 = 10
    num2 = 5
    result = num1 + num2
    print(result)  # Output: 15

  2. Subtraction (-):

    num1 = 10
    num2 = 5
    result = num1 - num2
    print(result)  # Output: 5

  3. Multiplication (*):

    num1 = 10
    num2 = 5
    result = num1 * num2
    print(result)  # Output: 50

  4. Division (/):

    num1 = 10
    num2 = 5
    result = num1 / num2
    print(result)  # Output: 2.0 (Note: In Python 3, division between integers returns a float)

Keep in mind that the division operation (/) between integers in Python 3 will return a floating-point number even if the result is a whole number. If you want integer division (floor division) that returns the quotient as an integer, you can use the double-slash operator (//):

num1 = 10
    num2 = 5
    result = num1 // num2
    print(result)  # Output: 2

These examples showcase how basic arithmetic operations can be performed using integers in Python, producing results that follow the rules of mathematics.

Modulus in Integers

Integer division and modulus are two useful operations when working with integers in Python.

  1. Integer Division (//): Integer division returns the quotient of a division operation, discarding the fractional part. It performs division and rounds down to the nearest whole number.

    num1 = 17
    num2 = 5
    quotient = num1 // num2
    print(quotient)  # Output: 3 (17 divided by 5 is 3 with a remainder)

  2. Modulus (%): The modulus operator returns the remainder of a division operation. It calculates what's left over after performing division.

    num1 = 17
    num2 = 5
    remainder = num1 % num2
    print(remainder)  # Output: 2 (17 divided by 5 leaves a remainder of 2)

The combination of integer division and modulus can be used to break down a number into its quotient and remainder parts. For example:

num = 17
    divisor = 5
    
    quotient = num // divisor
    remainder = num % divisor
    
    print("Quotient:", quotient)   # Output: Quotient: 3
    print("Remainder:", remainder) # Output: Remainder: 2

In this example, 17 divided by 5 gives a quotient of 3 and a remainder of 2. These operations are particularly useful for tasks like evenly distributing items, checking divisibility, or calculating time intervals.

  • Float Data Type:

    The float data type is used to represent decimal numbers in programming. Unlike integers, which only represent whole numbers, floats can represent numbers with fractional parts. In programming, floats are sometimes referred to as "floating-point numbers" because the decimal point "floats" to represent values at different scales.

Floats are essential for working with real-world values that are not whole numbers, such as measurements, monetary amounts, and scientific calculations. They're stored in a way that includes both a sign, a significant (or mantissa) part, and an exponent, allowing for a wide range of values.

Here's a simple example of using float data type in Python:

value1 = 3.14
    value2 = 0.25
    result = value1 + value2
    print(result)  # Output: 3.39

In this example, value1 and value2 are float variables containing decimal numbers. The result variable stores the sum of these values, and when printed, it displays the floating-point result.

While floats are powerful for representing a wide range of decimal values, it's important to note that they have limitations due to the way computers store them internally. Some operations can lead to small precision errors, especially when performing extensive calculations. In cases where precision is critical, specialized libraries or data types may be used to mitigate such issues.

3. String Data Type

Strings are a fundamental data type in programming used to represent sequences of characters. A character is a single element that can be a letter, digit, punctuation mark, space, or any other symbol. Strings allow programmers to work with and manipulate textual data.

Here's how the concept of strings as sequences of characters works:

  1. Sequence of Characters: A string is essentially a sequence of characters. Each character in the string is associated with a specific position, called an index. The characters can be letters, digits, symbols, or even special characters like newline and tab.

  2. Immutable: In many programming languages, strings are immutable, meaning once created, they cannot be changed directly. If you want to modify a string, you usually create a new string with the desired changes.

  3. String Literal: Strings are usually defined using double quotes ("") or single quotes (''). For example:

    • "Hello, world!" or 'Hello, world!'

  4. Concatenation: You can combine strings using concatenation. Concatenation joins two or more strings together to create a new string.

    greeting = "Hello"
    name = "Alice"
    full_greeting = greeting + ", " + name

  5. Indexing and Slicing: You can access individual characters within a string using their indices. Indices usually start from 0. You can also use slicing to extract portions of a string.

    text = "Python"
    first_character = text[0]  # 'P'
    sub_string = text[1:4]      # 'yth'

  6. Length: The length of a string is the number of characters it contains. This is commonly obtained using the len() function.

    message = "Hello, world!"
    length = len(message)  # 13

  7. Escape Sequences: Strings can include special characters using escape sequences. For example, "\n" represents a newline character, and "\t" represents a tab character.

  8. String Methods: Programming languages provide various built-in methods for manipulating strings. These methods allow you to convert cases, split strings, find substrings, replace characters, and more.

Strings are essential for dealing with textual information in programming, whether it's user input, file content, or any form of communication within a program. Understanding how strings are treated as sequences of characters helps programmers effectively manage and manipulate textual data.

Certainly! Let's dive deeper into string indexing, slicing, and concatenation:

  1. String Indexing: String indexing allows you to access individual characters within a string using their position, or index. In many programming languages, indices start from 0 for the first character and increase incrementally.

    message = "Hello, world!"
    first_character = message[0]   # 'H'
    second_character = message[1]  # 'e'

    You can also use negative indices to count from the end of the string:

    last_character = message[-1]    # '!'
    second_last_character = message[-2]  # 'd'

  2. String Slicing: String slicing allows you to extract a portion of a string by specifying a range of indices. The format is string[start:end], where start is inclusive and end is exclusive.

    message = "Hello, world!"
    hello = message[0:5]   # 'Hello'
    world = message[7:12]  # 'world'

    If you omit the start index, it defaults to 0. If you omit the end index, it defaults to the end of the string.

    greeting = message[:5]   # 'Hello'
    exclam = message[13:]    # '!'

  3. String Concatenation: String concatenation involves combining two or more strings into a single string. This is done using the + operator.

    first_name = "John"
    last_name = "Doe"
    full_name = first_name + " " + last_name  # 'John Doe'

    Strings can also be concatenated with other data types, but you might need to convert non-string data to strings first.

    age = 30
    info = "Name: " + full_name + ", Age: " + str(age)  # 'Name: John Doe, Age: 30'

These concepts are fundamental for working with strings in programming. By using indexing, slicing, and concatenation effectively, you can manipulate and extract relevant information from strings, making them a versatile tool for text processing and manipulation tasks.

  • Common string methods (.lower(), .upper(), .split(), .strip(), etc.).

Some common string methods that are widely used for string manipulation:

  1. .lower() and .upper():

    • .lower(): Converts all characters in a string to lowercase.
    • .upper(): Converts all characters in a string to uppercase.
    text = "Hello, World!"
    lower_text = text.lower()  # 'hello, world!'
    upper_text = text.upper()  # 'HELLO, WORLD!'

  2. .split():

    • .split(): Splits a string into a list of substrings based on a delimiter. The default delimiter is space.
    sentence = "This is a sample sentence."
    words = sentence.split()  # ['This', 'is', 'a', 'sample', 'sentence.']

  3. .strip() and .rstrip():

    • .strip(): Removes leading and trailing whitespace characters (spaces, tabs, newlines).
    • .rstrip(): Removes only trailing whitespace.
    input_text = "   Hello, World!   "
    clean_text = input_text.strip()  # 'Hello, World!'

  4. .replace():

    • .replace(old, new): Replaces occurrences of a substring with another substring.
    text = "I like apples and apples are tasty."
    new_text = text.replace("apples", "oranges")  # 'I like oranges and oranges are tasty.'

  5. .find() and .index():

    • .find(substring): Finds the index of the first occurrence of a substring. Returns -1 if not found.
    • .index(substring): Similar to .find(), but raises an error if not found. 
    message = "Hello, World!"
    index1 = message.find("World")  # 7
    index2 = message.index("Earth")  # Raises a ValueError since "Earth" is not in the string

  6. .startswith() and .endswith():

    • .startswith(prefix): Checks if a string starts with a specified prefix.
    • .endswith(suffix): Checks if a string ends with a specified suffix.
    filename = "document.txt"
    starts_with = filename.startswith("doc")  # True
    ends_with = filename.endswith(".pdf")  # False

These are just a few examples of the many string methods available in programming languages. By using these methods, you can efficiently perform various operations on strings, such as modifying their case, splitting them into parts, removing unwanted characters, and more.

4. Boolean Data Type

The Boolean data type is a fundamental data type in programming that represents two distinct values: True and False. Boolean values are used to express the truth or falsehood of statements, conditions, or comparisons within a program's logic.

Here's a brief explanation of each value:

  • True: Represents a condition or statement that is considered true or valid.
  • False: Represents a condition or statement that is considered false or invalid.

Boolean values are essential for decision-making and control flow in programming. They allow you to write code that performs different actions based on whether certain conditions are true or false. For instance, you might use Boolean values to control whether a loop continues running, to determine if a user has successfully logged in, or to check if a certain criteria is met before executing a certain block of code.

Boolean values can be used in combination with comparison and logical operators to create complex conditions and expressions that drive the behavior of your programs.

  • Comparison operators (==, !=, >, <, >=, <=) and their use.

Comparison operators are used to compare two values and determine the relationship between them. They return Boolean values (True or False) based on whether the comparison is true or false. Here are the commonly used comparison operators:

  1. Equal to (==): Checks if two values are equal.

    x = 5
    y = 5
    result = x == y  # True

  2. Not equal to (!=): Checks if two values are not equal.

    x = 10
    y = 20
    result = x != y  # True

  3. Greater than (>): Checks if the value on the left is greater than the value on the right.

    x = 15
    y = 10
    result = x > y  # True

  4. Less than (<): Checks if the value on the left is less than the value on the right.

    x = 5
    y = 8
    result = x < y  # True

  5. Greater than or equal to (>=): Checks if the value on the left is greater than or equal to the value on the right.

    x = 20
    y = 20
    result = x >= y  # True

  6. Less than or equal to (<=): Checks if the value on the left is less than or equal to the value on the right.

    x = 15
    y = 20
    result = x <= y  # True

Comparison operators are used extensively in programming to create conditions, make decisions, and control the flow of code. They're commonly used in if statements, loops, and various other control structures to determine whether certain actions should be taken based on the relationship between two values.

  • Logical operators (and, or, not) and their applications in conditions.

Logical operators are used to combine or manipulate Boolean values in conditions to create more complex expressions. They allow you to build conditions that involve multiple comparisons and determine the overall truth value of those comparisons. Here are the three main logical operators: and, or, and not.

  1. and Operator: The and operator returns True if both operands are True, and False otherwise.

    x = 10
    y = 20
    result = x > 5 and y < 30  # True (both conditions are true)

    The and operator is often used when you want to make sure multiple conditions are satisfied before taking an action.

  2. or Operator: The or operator returns True if at least one of the operands is True, and False if both operands are False.

    x = 10
    y = 5
    result = x > 15 or y > 3  # True (at least one condition is true)

    The or operator is useful when you want to take an action if any of multiple conditions is met.

  3. not Operator: The not operator returns the opposite of the Boolean value of its operand. If the operand is True, not returns False, and vice versa.

    x = 5
    result = not x > 10  # True (5 is not greater than 10)

    The not operator is often used to negate a condition or invert the truth value of a Boolean expression.

Logical operators are widely used in conditions to create more sophisticated decision-making logic. They allow you to combine multiple conditions and control the flow of your code based on various combinations of truth values. For example, you might use and to ensure that multiple conditions are met before taking action, use or to provide alternatives, and use not to invert the result of a condition.

5. Lists and Tuples

  • Lists:

    Lists are a specific data structure in programming that differ from other data types like integers, strings, floats, and Booleans. Here's how lists differentiate from other data types:

  1. Collection of Items:

    • Lists: A list is an ordered collection of items, where each item can be of any data type, including other lists. Lists allow you to store and organize multiple values in a single variable.
    • Other Data Types: Other data types represent individual values. For example, integers represent whole numbers, strings represent text, floats represent decimal numbers, and Booleans represent true/false values.

  2. Mutable:

    • Lists: Lists are mutable, meaning you can change, add, or remove elements after the list is created. You can modify items within the list without needing to create a new list.
    • Other Data Types: Most other data types, like integers, strings, floats, and Booleans, are immutable, meaning their values cannot be changed once they are created. You create new instances to represent different values.

  3. Ordered:

    • Lists: Lists maintain the order of elements, which means the position of each item in the list matters.
    • Other Data Types: Some data types, like dictionaries and sets, may not necessarily maintain a specific order.

  4. Use of Square Brackets:

    • Lists: Lists are defined using square brackets []. Elements are separated by commas.
    • Other Data Types: Other data types have their own syntax for definition. For example, integers, floats, and Booleans are simple values, while strings are defined using single or double quotes.

  5. Iteration and Indexing:

    • Lists: Lists can be iterated over using loops and accessed using indexing. You can access specific elements by their position (index) in the list.
    • Other Data Types: While other data types might be iterable (strings, for example), they are not typically accessed using indexes in the same way lists are.

  6. Flexible Size:

    • Lists: Lists can grow or shrink in size as you add or remove elements.
    • Other Data Types: Most other data types have a fixed size once they're created.

Overall, lists are versatile data structures that allow you to work with collections of items in a flexible and dynamic way. They are commonly used for storing related data, managing collections, and implementing various algorithms and data manipulation tasks.

Creating a List:

You can create a list by enclosing items in square brackets [], separated by commas.

fruits = ["apple", "banana", "orange", "grape"]
    print(fruits)  # Output: ['apple', 'banana', 'orange', 'grape']

Accessing List Elements:

You can access list elements using their indices.

fruits = ["apple", "banana", "orange", "grape"]
    
    first_fruit = fruits[0]
    print(first_fruit)  # Output: apple
    
    second_fruit = fruits[1]
    print(second_fruit)  # Output: banana
    
    third_fruit = fruits[2]
    print(third_fruit)  # Output: orange

You can also use negative indices to count from the end of the list:

last_fruit = fruits[-1]
    print(last_fruit)  # Output: grape
    
    second_last_fruit = fruits[-2]
    print(second_last_fruit)  # Output: orange

Modifying List Elements:

Lists are mutable, so you can change their elements after creation.

  • Assign a new value to a specific index to modify an element:
fruits[1] = "cherry"
    print(fruits)  # Output: ['apple', 'cherry', 'orange', 'grape']
  • Use slicing to modify multiple elements at once:
fruits[1:3] = ["cherry", "pear"]
    print(fruits)  # Output: ['apple', 'cherry', 'pear', 'grape']

fruits.append("kiwi")

del fruits[1]
    print(fruits)  # Output: ['apple', 'melon', 'pear', 'grape', 'kiwi', 'peach', 'plum']
  • Use the .remove() method to remove elements by value:
fruits.remove("orange")
    print(fruits)  # Output: ['apple', 'melon', 'pear', 'grape', 'kiwi', 'peach', 'plum']

  • 6. Tuples:

    • Immutability of tuples and use cases.

Tuples are similar to lists in Python, but they have a key difference: tuples are immutable, while lists are mutable. This means that once you create a tuple, you cannot modify its elements. Here's a closer look at the immutability of tuples and their use cases:

Immutability of Tuples:

  1. Cannot Modify Elements: Once a tuple is created, you cannot change, add, or remove elements. This property makes tuples immutable data structures.

  2. Preservation of Data: Because tuples are immutable, the data they hold remains constant throughout their lifetime. This can be useful when you want to ensure that the data doesn't accidentally get modified.

Use Cases for Tuples:

  1. Data Integrity: Tuples are often used to store related pieces of data that belong together and shouldn't change. For example, you might use a tuple to store coordinates (x, y) or dates (year, month, day).

  2. Dictionary Keys: Tuples can be used as keys in dictionaries because of their immutability. Lists, being mutable, cannot be used as dictionary keys.

  3. Function Return Values: Functions can return multiple values using a tuple. Since tuples are immutable, they ensure the integrity of the returned data.

  4. Unpacking: Tuples are often used to unpack multiple values returned by functions or when iterating over sequences.

  5. Hashable for Sets: Tuples are hashable, making them suitable for use as elements in sets. Lists, being mutable, are not hashable and cannot be elements in sets.

  6. Database Records: Tuples are used to represent individual records in databases, where each field value is immutable.

Example use case:

# Using a tuple to represent a point's coordinates
    point = (3, 5)  # Immutable tuple
    x, y = point    # Unpacking the tuple
    
    # Using a tuple to return multiple values from a function
    def divide(x, y):
        quotient = x // y
        remainder = x % y
        return quotient, remainder
    
    result = divide(17, 5)
    quotient, remainder = result

In summary, tuples are immutable data structures that provide data integrity and are suitable for cases where you want to store related data that should not change. Lists are better suited for scenarios where you need to modify, add, or remove elements.

    second_last_fruit = fruits[-2]
    print(second_last_fruit)  # Output: orange

Tuple elements are accessed in the same way as list elements, but remember that tuples are immutable, so you cannot modify their elements after creation.

6. Dictionaries

Dictionaries in Python are a versatile and powerful data structure used to store and manage collections of data in the form of key-value pairs. Each element in a dictionary is made up of two main components: a key and its associated value. The key acts as a label or identifier, while the value holds the actual data associated with that key. This relationship allows for efficient and fast retrieval of data based on its unique key.

Here's a breakdown of how dictionaries work:

  • Keys: Keys are unique and immutable (cannot be changed after creation) identifiers that serve as the labels for the associated values. They are typically strings, numbers, or tuples.

  • Values: Values can be of any data type: strings, numbers, lists, other dictionaries, or even more complex objects.

  • Key-Value Pairs: In a dictionary, each key is associated with a specific value. Together, they form a key-value pair.

  • Unordered: Dictionaries are unordered collections, meaning that the elements are not stored in a specific order. However, starting from Python 3.7, dictionaries maintain the insertion order of their elements.

Here's an example of a dictionary:

# Creating a dictionary of person's information
    person = {
        "name": "Alice",
        "age": 30,
        "city": "New York"
    }

In this example, "name", "age", and "city" are keys, and "Alice", 30, and "New York" are their corresponding values.

Dictionaries are useful for various scenarios:

  1. Fast Data Retrieval: Dictionaries provide fast lookups based on keys. You can quickly access the value associated with a specific key without needing to iterate through the entire collection.

  2. Mapping Relationships: Dictionaries are often used to map one value to another. For example, you could use a dictionary to map words to their meanings or to store configurations.

  3. Storing Structured Data: Dictionaries allow you to store structured data without requiring fixed positions or indices. This is useful when the data doesn't naturally fit into a linear sequence.

  4. Managing Counters: Dictionaries can be used to maintain counters and frequency distributions. Keys represent elements, and values represent their occurrence frequencies.

word_counts = {
        "apple": 5,
        "banana": 3,
        "orange": 7
    }

Overall, dictionaries provide a flexible and efficient way to organize and manipulate data based on key-value relationships.

  • Dictionary methods (.keys(), .values(), .items()) and how to manipulate dictionaries.

Certainly! Here are some common dictionary methods in Python, along with explanations of their usage and how to manipulate dictionaries:

.keys() Method:

  • Returns a view of all the keys in the dictionary.
  • The result is a "view object" that can be converted to other data types, like a list or set.
person = {
        "name": "Alice",
        "age": 30,
        "city": "New York"
    }
    
    keys = person.keys()
    print(keys)  # Output: dict_keys(['name', 'age', 'city'])
    
    # Convert the view to a list
    key_list = list(keys)
    print(key_list)  # Output: ['name', 'age', 'city']

.values() Method:

  • Returns a view of all the values in the dictionary.
  • Similar to .keys(), the result is a view object that can be converted to other data types.
person = {
        "name": "Alice",
        "age": 30,
        "city": "New York"
    }
    
    values = person.values()
    print(values)  # Output: dict_values(['Alice', 30, 'New York'])
    
    # Convert the view to a list
    value_list = list(values)
    print(value_list)  # Output: ['Alice', 30, 'New York']

.items() Method:

  • Returns a view of all the key-value pairs in the dictionary as tuples.
  • The result is a view object that can also be converted to other data types.
person = {
        "name": "Alice",
        "age": 30,
        "city": "New York"
    }
    
    items = person.items()
    print(items)  # Output: dict_items([('name', 'Alice'), ('age', 30), ('city', 'New York')])
    
    # Convert the view to a list of tuples
    item_list = list(items)
    print(item_list)  # Output: [('name', 'Alice'), ('age', 30), ('city', 'New York')]

Manipulating Dictionaries: Dictionaries can be manipulated using various methods to add, update, and remove elements:

  • Adding a new key-value pair:
person = {
        "name": "Alice",
        "age": 30
    }
    
    person["city"] = "New York"
    print(person)  # Output: {'name': 'Alice', 'age': 30, 'city': 'New York'}
  • Updating a value:
person["age"] = 31
    print(person)  # Output: {'name': 'Alice', 'age': 31, 'city': 'New York'}
  • Removing a key-value pair:
del person["age"]
    print(person)  # Output: {'name': 'Alice', 'city': 'New York'}
  • Using the .pop() method:
removed_age = person.pop("age")
    print(removed_age)  # Output: 31
    print(person)  # Output: {'name': 'Alice', 'city': 'New York'}

These methods and techniques allow you to manipulate and work with dictionaries effectively, making them a versatile tool for storing and organizing data.
